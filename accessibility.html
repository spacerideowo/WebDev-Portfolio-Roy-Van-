Accessibility Features

This website follows Web Content Accessibility Guidelines to improve accessibility. I used semantic HTML elements like <header>, <main>, and <footer>. All images use alt text. Colors are high contrast, navigation is keyboard-friendly, and font sizes are readable across screen sizes.

This website is built with accessibility in mind. I have ensured that all visual elements are accompanied by meaningful alternative text to support screen readers and users with visual impairments.
